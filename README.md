# 🚀 Meu Primeiro Backend com Node.js

Este é meu primeiro projeto de backend utilizando Node.js e Express.

## 📚 Tecnologias utilizadas

* Node.js
* Express

## ⚙️ Funcionalidades

* Servidor HTTP básico
* Primeira rota funcionando (`/`)

## ▶️ Como rodar o projeto

```bash
npm install
node index.js
```

Acesse no navegador:
http://localhost:3000
